# Comandos Unix

## Cambiar configuración de teclado (modo grafico)

### Instalar x11-xkb-utils

``` shell
sudo apt install x11-xkb-utils
```

``` shell
setxkbmap -layou -model pc105
```

### Por linea de comandos

#### Teclado en español

``` shell
sudo loadkeys es
```

#### Teclado EEUU

``` shell
sudo loadkeys us
```

#### Nombre del Host

Nombre del host

``` shell
hostname
```

``` shell
hostname
```

Ip del Host

``` shell
uname -a
```

Cambiar nombre del host

``` shell
sudo hostname nuevo_host_name
```
