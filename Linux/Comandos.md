# Comandos Unix

## Cambiar configuración de teclado (modo grafico)

### Instalar x11-xkb-utils

```linux
sudo apt install x11-xkb-utils
```

```linux
setxkbmap -layou -model pc105
```

### Por linea de comandos

#### Teclado en español

```linux
sudo loadkeys es
```

#### Teclado EEUU

```linux
sudo loadkeys us
```
